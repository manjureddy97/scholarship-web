# Scholarship Application System (Dummy, Next.js 14)

Polished, client-ready UI with dummy data and full flow: login → dashboard → new application → evaluation → admin reports.

## Run
```bash
pnpm i
pnpm dev
```
Open http://localhost:3000

## Tech
- Next.js 14 (App Router, TS), TailwindCSS
- React Hook Form + Zod for validation
- Context auth (Redux-ready structure can be added under /store)
- Dummy JSON data under /data
